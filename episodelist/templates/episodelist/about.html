Список эпизодов

Работает только для форума King's Cross.

How it works

На отдельной странице этого сайта с помощью API форумов соибирается список всех тем, на которые подписан определенный пользователь. ID пользователя нужно передать в ссылке.

Пример: https://viper-frpg.ovh/episodelist/2188

Если передать несколько ID, то список будет составлен из всех тем, на которые подписан хотя бы один из пользователей.

Пример: https://viper-frpg.ovh/episodelist/2184,2188

What to do

Если вы хотите сделать автоматически обновляющийся список эпизодов своего каста, вам нужно сделать три вещи:

  • Сложная часть: уговорите всех игроков в касте начать подписываться на свои эпизоды
  • В коде ниже замените id в ссылке id игроков из вашего каста
  • Вставьте скрипт на видное место и гордитесь
